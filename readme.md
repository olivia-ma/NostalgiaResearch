The files are called in the order -
1. [Data Extraction](data_extraction.py) - Extracts the datasets
2. [Merging Datasets](merge_and_initial_analysis.py) - Cleans and merges the dataset
3. [Labels](labels.py) - Adds labels to the reviews
4. [Features](features.py) - Add the features for the model stage
5. [Model](binary_models.py) - Model
6. [Insights](insights.py) - Insights on the model
